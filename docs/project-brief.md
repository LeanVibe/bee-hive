# Project Brief: Bee Hive

## Problem
- **Who is the user?** Engineering teams managing multi-agent systems
- **What pain is urgent + expensive?** Need real-time operational dashboard for multi-agent orchestration

## Promise (Painkiller)
- In 1 sentence: Modern FastAPI backend with Lit + Vite PWA for real-time operational dashboards and autonomous multi-agent orchestration
- What is the narrow wedge? Real-time dashboard + WebSocket updates + unified CLI

## ICP
- **Role:** Engineering teams, DevOps engineers
- **Context:** Managing multi-agent systems and need visibility
- **Budget:** Enterprise tool pricing ($199-799/month)
- **Trigger:** Need operational visibility into agent workflows

## MVP Scope (14 days)
- **Critical path:**
  - Real-time dashboard
  - WebSocket updates
  - System monitoring
  - Unified CLI (hive command)
- **Out of scope:**
  - Advanced analytics
  - Enterprise SSO

## Pricing Hypothesis
- **Price:** $199/month for teams, $799/month for enterprise
- **Packaging:** Team: basic dashboard, Enterprise: advanced analytics

## Risks / Assumptions
- Top 3 assumptions to validate first:
  1. Teams need real-time operational dashboards
  2. WebSocket-based updates are preferred
  3. Unified CLI (hive) is valuable

## Status
- **Intake Date:** 2026-01-20
- **Stack:** Python FastAPI, Lit + Vite PWA, PostgreSQL, Redis
- **Repository:** Active git repository
- **Features:** Real-time dashboard, WebSocket updates, unified CLI, system monitoring, technical debt remediation tools
