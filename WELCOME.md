## ⚡ Quick Start (5 minutes)

**For the impatient**: Get autonomous development working now.

```bash
# 1. Clone and setup (one command)
git clone https://github.com/LeanVibe/bee-hive.git && cd bee-hive && ./setup-fast.sh

# 2. Add your API key (required)
echo "ANTHROPIC_API_KEY=your_key_here" >> .env.local

# 3. Watch autonomous development in action
python scripts/demos/autonomous_development_demo.py
```
